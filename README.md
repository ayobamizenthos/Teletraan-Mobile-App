**Teletraan Mobile**

Mobile app for the Avzdax surveillance platform.

View web version at [teletraan-mobile.vercel.app](https://teletraan-mobile.vercel.app)

**Features**

- Live camera dashboard with quad and single view
- Fullscreen camera with swipe, zoom, mute and snapshot capture
- Per-camera alert history and threat scoring
- Share incident media with caption in one intent
- Global search across cameras, alerts and locations
- Notifications grouped by day with deep links to incidents

**Tech stack**

- Expo, React Native, TypeScript
- Expo Router, Reanimated, react-native-svg
- Zustand for state
